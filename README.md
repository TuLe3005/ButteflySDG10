# SDG10 Reduce Inequality

**Version:** 1.0  
**Date:** 27/3/2026

## Usage Notes
This program features three main interactions:
- **Mouse holding:** Hold the mouse button to interact with the animation.
- **Arrow Up key holding:** Hold the up arrow key to expand opportunities.
- **Arrow Down key holding:** Hold the down arrow key to constrain opportunities.

## Installation Instructions
- Start a web server in the root directory of this project.
- Open `index.html` in your browser to see and hear the results.
- No dependencies or plug-in installation is required.
